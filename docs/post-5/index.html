Surface

Como dice el titulo, en este momento nos enseñan a crear dos lineas en un sketch, que despues se extruyen. para utilizarlas como corte en una figura hecha anteriormente. Despues estas a esta figura se le hace un offset surface para luego ser escaladas. Para finalmente, se realiza lofts entre las superficies sueltas, utilizando también surface stich, el cual funciona pomo un join en las superficies. Para terminar utilizando fillet.
