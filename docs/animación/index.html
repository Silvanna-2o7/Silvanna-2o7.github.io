Clip Studio

Hola, en este post les enseñare a usar clip studio para crear una animación y que formas se pueden descargar

Puedes descargar la animacion de varias formas:

  • Gif
  • Movie
  • Animate Sticker (APNG)
  • Imagen Sequence

GIF

Este es el formato mas simple y que deja la animación en loop sin necesidad de hacerlo por ti mismo.

MOVIE

Al descargarlo como movie, queda en (.avi), asi que para subirlo a alguna pag web, quizas deberás cambiarlo con premier o otra aplicacion a (.mp4)

formato avi

formato mp4

APNG

APNG o mejor llamado animated portable network graphics es un png que tiene las especificaciones que permite hacerla animación, trabaja parecido a un GIF, solo que soporta imagenes de 24-bit y 8-bits transparency que el formato gif no habilita.

IMAGEN SEQUENCE

Con Imagen sequence puedes guardar tu animacion fotograma por fotograma, lo cual te sirve para subirla a otras aplicaciones de videos, etc.
